# Hangman Game

This is a terminal-based hangman game that includes multiple categories, hints, and maintains high scores.

## Features
- Choose from various categories
- Hints for each word
- High score tracking

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/hangman.git
   cd hangman
   ```
2. Install the required packages:
   ```bash
   pip install colorama
   ```

## Usage
Run the game with:
```bash
python hangman.py
```
